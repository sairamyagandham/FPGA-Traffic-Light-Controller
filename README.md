# FPGA-Based Traffic Light Controller with Emergency Vehicle Priority

## Overview

This project implements a Traffic Light Controller using Verilog HDL and Finite State Machine (FSM) concepts.

The controller manages traffic at a two-road intersection and provides priority access to emergency vehicles. It also supports adaptive timing based on vehicle density.

## Features

* FSM-Based Traffic Light Control
* Emergency Vehicle Priority Override
* Vehicle Density-Based Green Time Extension
* Verilog HDL Design
* Functional Verification using Testbench
* FPGA Compatible Design

## Tools Used

* Verilog HDL
* EDA Playground
* EPWave / GTKWave
* Vivado (Optional FPGA Implementation)
